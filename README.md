# Unnamed Incremental Game

A dungeon-themed idle clicker built with [SRCL](https://github.com/internet-development/www-sacred) terminal-aesthetic React components.

## Play

**[Live demo](https://fearful-avoidant.github.io/unnamed-incremental-game-pages/dungeon.html)**

## Development

```sh
npm install
npm run dev
```

Go to `http://localhost:10000/dungeon`.

## Deploy

Pushes to `main` automatically build and deploy to GitHub Pages via `.github/workflows/deploy.yml`.

## Stack

- [Next.js](https://nextjs.org/) (static export)
- [SRCL / sacred computer](https://github.com/internet-development/www-sacred) components
